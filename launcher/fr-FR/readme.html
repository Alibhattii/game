1)  Configuration nécessaire
CONFIGURATION MINIMALE REQUISE:
Windows XP SP3, Vista SP2, 7
Processeur Intel Pentium IV 2.4GHz ouéquivalent
1 GO RAM
8 GO d’espace libre sur le disque dur
Carte vidéo NVIDIA GeForce 6600 ou AMD/ATI Radeon x1300. Pixel/Vertex Shader 3.0 et une carte vidéo VRAM 128 MB DirectX 9.0c compatible sont requis.
Carte son compatible DirectX 9.0c
DirectX 9.0c ou version plus récente (logiciel inclus sur le Disque)
800 x 600 résolution du moniteur

CONFIGURATION RECOMMANDEE :
Windows XP SP3, Vista SP2, 7
Processeur Intel Core2 Duo 2,0GHz
2 GO RAM
8 GO d’espace libre sur le disque dur
NVIDIA GeForce 7900, AMD/ATI Radeon HD2400 ouversion plus récente. Pixel/Vertex Shader 3.0 et une carte vidéo VRAM 512 Mo DirectX 9.0c compatible sont requis.
Carte son compatible DirectX 9.0c
DirectX 9.0c ou version plus récente (logiciel inclus sur le Disque)
1280 x 720 résolution du moniteur

COMPATIBLE AVEC LES CIRCUITS :
NVIDIA - GeForce GTX285, 9800, 8800, 8600, 7950, 7900, 7800, 7300, 6800, 6600, 6200
AMD/ATI - Radeon HD5850, HD5770, HD4870, HD4650, HD3870, HD2600, HD2400, X1900, X1650, X1600, X1300

Il est parfois nécessaire de mettre à jour
vos pilotes logiciels avant de commencer à jouer.
