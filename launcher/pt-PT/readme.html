1)  Requisitos do sistema
REQUISITOS MÍNIMOS DO SISTEMA:
Windows XP SP3/Vista SP2/7
Intel Pentium IV 2.4GHz ou processador equivalente
1GB de memória RAM
8GB de espaço livre no disco rígido
DirectX 9.0c que seja compatível com a placa de vídeo. 128MB Pixel Shader 3.0 (NVIDIA GeForce 6600 ou placa de vídeo ATI/AMD Radeon x1300)
DirectX 9.0c que seja compatível com a placa de som
DirectX 9.0c ou superior (está incluído no DVD)
Teclado compatível com o sistema operativo Windows
Resolução do monitor:800x600

CONFIGURAÇÃO DE SISTEMA RECOMENDADA:
Windows XP SP3/Vista SP2/7
Intel Core2 Duo 2.0GHz ou processador equivalente
2GB de memória RAM
8GB de espaço livre no disco rígido
DirectX 9.0c que seja compatível com a placa de vídeo. 512MB Pixel Shader 3.0 (ATI/AMD Radeon HD2400 ou NVidia GeForce 7900 ou superior)
DirectX 9.0c que seja compatível com a placa de som
DirectX 9.0c ou superior (está incluído no DVD)
Gamepad compatível com o sistema operativo Windows
Resolução do monitor: 1280x 720

CHIPSETS SUPORTADOS:
NVIDIA - GeForce GTX285, 9800, 8800, 8600, 7950, 7900, 7800, 7300, 6800, 6600, 6200
AMD/ATI - Radeon HD5850, HD5770, HD4870, HD4650, HD3870, HD2600, HD2400, X1900, X1650, X1600, X1300
Intel - HD Graphics 2000, HD Graphics 3000

Antes de jogar talvez tenhas que actualizar os controladores (drivers) do teu hardware.

As versões destes chipsets para portáteis talvez funcionem correctamente mas NÃO são suportadas pelo fabricante.
