1)  Requisiti di sistema
REQUISITI MINIMI DI SISTEMA:
Windows XP SP3, Vista SP2, 7
Intel Pentium IV a 2.4 GHz o processore equivalente
1 GB di RAM
8 GB di spazio libero sul disco rigido
Scheda video NVIDIA GeForce 6600 o AMD/ATI Radeon x1300. La scheda grafica deve essere compatibile con DirectX9.0c, Pixel/Vertex Shader 3.0 o superiore, e dotata di 128MB o più di VRAM.
Scheda audio compatibile con DirectX 9.0c
DirectX 9.0c o superiore (incluso nel Disco 1)
Tastiera compatibile con Windows
Risoluzione del monitor 800 x 600

REQUISITI DI SISTEMA CONSIGLIATI:
Windows XP SP3, Vista SP2, 7
Processore Intel Core2 Duo a 2.0 GHz
2 GB di RAM
8 GB di spazio libero sul disco rigido
Scheda video NVIDIA GeForce 7900, AMD/ATI Radeon HD2400 o superiore. La scheda grafica deve essere compatibile con DirectX9.0c, Pixel/Vertex Shader 3.0 o superiore, e con 512MB o più di VRAM.
Scheda audio compatibile con DirectX 9.0c
DirectX 9.0c o superiore (incluso nel Disco 1)
Gamepas USB compatibile con Windows
Risoluzione monitor 1280 x 720

CHIPSET SUPPORTATI:
NVIDIA - GeForce GTX285, 9800, 8800, 8600, 7950, 7900, 7800, 7300, 6800, 6600, 6200
AMD/ATI - Radeon HD5850, HD5770, HD4870, HD4650, HD3870, HD2600, HD2400, X1900, X1650, X1600, X1300
Intel - HD Graphics 2000, HD Graphics 3000

Prima di giocare, è possibile che sia necessario aggiornare i driver hardware.

Le versioni per PC portatile di queste schede grafiche potrebbero funzionare ma non sono supportate.
